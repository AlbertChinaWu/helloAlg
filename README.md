## 关于本书

本书面向算法初学者，致力于构建一本开源免费、新手友好的数据结构与算法入门书。

- 全书采用动画图解，致力于结构化地讲解数据结构与算法知识，内容清晰易懂、学习曲线平滑；
- 算法源代码皆可一键运行，现已支持 Java, C++, Python, Go, JS, TS, C#, Swift 等语言；
- 提供章节讨论区，提问与评论一般能在两日内得到回复；

如果本书对您有所帮助，请点个 Star :star: 支持一下，谢谢！
